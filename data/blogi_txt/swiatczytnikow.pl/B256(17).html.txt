Microsoft, o którym przeciętni użytkownicy Windows mogą nie mieć dobrego zdania dba jednak o ludzi, którzy z technologii Microsoftu żyją – programistów, projektantów, wdrożeniowców. Dla nich też przygotowano kilka darmowych książek. Do niedawna tylko w PDF, a od 1 sierpnia także na czytniki. W formatach e-bookowych dostępne jest pięć pozycji:  Charles Petzold’s Programming Windows Phone 7: EPUB, MOBI Patrice Pelland, Pascal Paré, Ken Haines Moving to Microsoft Visual Studio 2010: EPUB, MOBI Ross Mistry, Stacia Misner Introducing Microsoft SQL Server 2008 R2: EPUB, MOBI Charlie Russel and Craig Zacker with the Windows Server Team at Microsoft Introducing Windows Server 2008 R2: EPUB, MOBI Katherine Murray’s Own Your Future, Update Your Skills with Resouces and Career Ideas from Microsoft: EPUB, MOBI Uwaga przy ściąganiu, pliki są duże, rzędu 50 MB. Microsoft na swoim blogu podaje obecnie 8 darmowych książek – reszta w PDF. Wszystko to są specjalistyczne pozycje, raczej nie przydadzą się tym, którzy mają styczność tylko z Windows i Wordem. Ale zawsze można rzucić okiem jak taka książka o programowaniu wygląda. Aczkolwiek tutaj się nie popisali i np. kod podają w postaci obrazków, a nie tekstu. Dziękuję Maciejowi Chojnackiemu za donos i linka do forum społeczności Windows Server System, gdzie temat pojawił się wcześniej.
