Informatycy i miłośnicy fantastyki, to chyba dwie najbardziej dopieszczane grupy czytelników e-booków. Nie tylko sporo tytułów jest do kupienia, ale także można pobrać wiele za darmo. O darmowych lekturach z Microsoftu wspominałem ponad dwa lata temu. Zaczęło się skromnie, bo od pięciu tytułów. Dzisiaj mamy już ich prawie 150! Propagowaniem książek z Microsoft Press zajmuje się na swoim blogu Eric Lingman, pracujący w firmie na stanowisku „Sales Excellence Manager”, cokolwiek by ono nie znaczyło. Ostatni zestaw e-booków udostępnił w czerwcu 2013. Tematyka bardzo różnorodna:  Office (np. Microsoft Office 365 for professionals and small businesses) SharePoint (np. Deployment guide for SharePoint 2013) SQL Server (np. A Hitchiker’s Guide to Microsoft StreamInsight Queries) System Center Visual Studio Web Development (np. Developing Modern Mobile Web Apps) Windows (np. Programming Windows 8 Apps with HTML, CSS, and JavaScript) Windows Azure (np. Create Your First Application: Node.js and Windows Azure)  Dla „zwykłego człowieka” nie ma chyba tutaj propozycji, ale dla ludzi zajmujących się technologiami MS zawodowo jest sporo dobrego – zarówno dla początkujących, jak i zaawansowanych. Co ważne – większość w trzech formatach – EPUB, MOBI oraz PDF – tylko niektóre jako PDF/XPS. W kolejnym artykule Ligman podał TOP50 najpopularniejszych darmowych książek z Microsoftu. Ograniczę się do pierwszej dziesiątki:  Windows 7 Power Users Guide Introducing Windows Server 2012 – RTM Edition! Introducing Microsoft SQL Server 2012 Programming Windows Phone 7 Introducing Microsoft SQL Server 2008 R2 Get started with SharePoint Server 2010 Office 365 – Connect and Collaborate virtually anywhere, anytime Understanding Microsoft Virtualization R2 Solutions Microsoft Office 2010 First Look Moving to Microsoft Visual Studio 2010    Zdradził też liczbę pobrań – otóż tylko do końca czerwca ściągnięto 3,5 miliona plików! Ta liczba poraża. Jasne, jak darmowe, to niektórzy ściągają wszystko, niezależnie czy ma to sens, ale chyba każdy wydawca marzyłby o takiej popularności. Firma nie udostępnia tych pozycji z dobrego serca, wszystko co wspomoże użytkowników oprogramowania Microsoftu jest dla niej dobre, bo zwyczajnie zachęca do kupna produktów tej firmy. Pytanie, na ile taka darmocha wpływa na przychody innych wydawnictw IT – bo przecież takie O’Reilly czy Peachpit również mają bogatą ofertę na temat Microsoftu. Wydaje mi się jednak, że to jest pożyteczne uzupełnienie – a zawodowcy chętnie zaopatrzą się również w książki niezależne od producenta. To akurat branża, w której wydatków na własną edukację nie trzeba uzasadniać. Czytaj dalej: Nook debiutuje w Polsce za pośrednictwem Microsoftu! Na razie rozczarowuje Darmocha od Nexto i Wydawnictwa MAG: Gwiezdny pył i Kamienna Ćma Pies Łańcuchowy – wydawnictwo literackie z książkami… za darmochę!
