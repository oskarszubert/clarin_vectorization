21.10.2011 Transportoid to bardzo przydatna aplikacja dla wszystkich tych którzy podróżują po mieście. Zawiera w sobie plany oraz rozkłady jazdy komunikacji miejskiej wraz z możliwością planowania podróży. Program w tej chwili obsługuje ponad 50 polskich aglomeracji i do tej pory był dostępny jedynie na telefony wyposażone w system Android.  Od dziś jednak również użytkownicy sprzętu z Windows Phone 7 mogą korzystać z wygodnego planowania podróży komunikacją miejską za pomocą beta wersji Transportoidu (nie wszystkie funkcje są jeszcze dostępne ale podejrzewam, że jest to tylko kwestia czasu) Należy podkreślić, że wersja na WP7 nie jest kalką wcześniejszego wydania – interfejs użytkownika został dopasowany do wytycznych Windows Phone Z bloga aplikacji #gallery-1 {                                 margin: auto; }                         #gallery-1 .gallery-item { float: left;                                 margin-top: 10px; text-align: center;                                 width: 25%; }                         #gallery-1 img { border: 2px solid #cfcfcf;                         } #gallery-1 .gallery-caption { margin-left: 0;                         } /* see gallery_shortcode() in wp-includes/media.php */ Jeśli ktoś z was nie miał okazji wcześniej zapoznać się z funkcjami Transportoida wymienię kilka najważniejszych cech tej aplikacji:  - Wygodne filtrowanie przystanków i linii  po nazwach (starczą często 2-3 pierwsze litery/cyfry) - Po znalezieniu szukanego przystanku lub linii starczy jedno “tapnięcie”, aby uzyskać rozkład z podświetlonym najbliższym odjazdem - Możliwość zdefiniowania przystanku początkowego i końcowego. Transportoid znajdzie odpowiednie środki komunikacji, przystanki przesiadki, godziny odjazdów i prawdopodobny czas trwania przejazdu. - Możliwość dodawania tras, przystanków i linii do ulubionych, co oszczędza ponownego wyszukiwania. - Aplikacja działa błyskawicznie, bo informacje składuje lokalnie.
