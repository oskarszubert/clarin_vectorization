Ewangelista nowych technologii i innowacji.   @iTomek   Email Tomasz Fiedoruk, 18 maja 2012, 10:18   0   Windows 8 Community Launch Studenci Rzeszowskiej Grupy IT oraz Rzeszowskiej Grupy .NET serdecznie zapraszają pasjonatów programowania i nie tylko na konferencję informatyczną, która odbędzie się w Sali V-1 Politechniki Rzeszowskiej. Windows 8 Community Launch to konferencja poświęcona w głównej mierze nowemu systemowi operacyjnemu Windows 8, jak również oprogramowaniu serwerowemu Windows Server 2012. Uczestnicy konferencji zapoznają się ze zmianami, jakie wnoszą nowe systemy oraz narzędziami służącymi do tworzenia aplikacji kompatybilnych z nowym interfejsem Metro UI. Podczas spotkania będzie można posłuchać o nowym systemie Windows 8 oraz Windows Server 2012 od strony programistycznej, jak i administracyjnej.
