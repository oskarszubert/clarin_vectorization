Social media flamethrower!   @priard   Email   Łukasz Ładecki, 19 czerwca 2010, 6:09   2 Windows Embedded Handheld OS – kolejny system mobilny Microsoftu Microsoft zapowiedział nowy system operacyjny dla urządzeń mobilnych, nowi nazwę: Windows Embedded Handheld OS. Nowy system pojawi się prawdopodobnie jeszcze w tym roku i jest pisany z myślą o rozwiązaniach biznesowych. Początkowo będzie oparty o Windows Mobile 6.5.3, ale już w drugiej połowie 2011 roku powstanie kolejna wersja tego systemu, tym razem oparta o Windows Phone 7. Tak jak pisałem nowy system będzie przygotowany dla klientów biznesowych – poprawione bezpieczeństwo danych, specjalnie przygotowane aplikacje i interfejs. Microsoft jest znany z wydawanie wielu wersji swoich systemów, np.:  Windows 7 Starter, Home Basic, Home Premium, Professional etc., co nieraz wyśmiewał Apple. Dla mnie taka polityka jest raczej śmieszna, wprowadza niepotrzebny zamęt i utrudnia wybór. W tej chwili kolejna wersja systemu przeznaczonego dla urządzeń mobilnych? – Trochę to dziwne, jaki jest cel takich zabiegów? Czemu one mają służyć? Jest Windows CE, Windows Mobile 6,5, Windows Phone 7 (celowo rozgraniczam te wersje dlatego, że nie są kompatybilne wstecz), a teraz będziemy mieli jeszcze Windows Embedded Handheld. Dobrze mieć wybór, ale tworzenie kolejnego systemu mobilnego na bazie poprzedniej wersji WM6.5 (w dodatku z interfejsem upodobnionym do WP7), to ma w czymś pomóc firmie? Rozumiem gdyby był to WP7 w dwóch wersjach: przeznaczonych dla firm i dla odbiorców indywidualnych, nie inaczej! Przykład powiedzmy firmy HTC, która bierze Androida i wydaje go z własną wersją interfejsu Sense, pokazuje że tworzenie kolejnych systemów jest niepotrzebne, wystarczy przygotować aplikacje i interfejs, ale to tylko moje zdanie. :)
