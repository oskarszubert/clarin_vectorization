Ewangelista nowych technologii i innowacji.   @iTomek   Email Tomasz Fiedoruk, 12 maja 2009, 8:11   9   Optymalizacja Windows 7 Jeśli chcesz modyfikować opcje i ustawienia w Windows 7, przyda Ci się do tego program XdN. Działa on na Windows XP, 2003, Vista, Server 2008 i oczywiście Windows 7. A co umożliwia sam program? Ustawienia dotyczą zarówno ustawienia wyświetlania się różnych elementów, optymalizacji podstawowych funkcji systemowych czy menu kontekstowego oraz wielu innych newralgicznych elementów naszego systemu. Disable/Enable ‘SendTo’ right-click menu. Allows you to adjust and tweak TCP/IP settings. Disable/Enable ZipFolders. Disable/Enable the Outlook Express/Windows Mail splash screen. Put the File menu above the Back/Forward buttons in IE7. Remove or Replace the Search box in IE7. Disable/Enable User Account Control prompts under Vista. Disable/Enable the “- Shortcut” text under Vista. Attempt to force Vista to better save folder views. Disable/Enable Windows Media Player Explorer context menus. Add/Remove “Take Ownership” to right-click menus under Vista. Disable/Enable the Documents item on the Classic Start Menu. Changes “Delete” to “Search…” on Recycle Bin. Turn off the annoying full screen “black” UAC prompt. Add/Remove the Network Connections icon to Control Panel Enable Classic Search in Explorer and Internet Explorer.
