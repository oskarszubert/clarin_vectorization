Pracuję na stanowisku specjalisty ds. e-PR w firmie Projekt PR. @lukasz_matusik   Email   Łukasz Matusik, 27 października 2010, 1:27   1   Service Pack 1 RC dla systemów Windows 7 i Windows Server 2008 Wczoraj informowaliśmy o wycieku Service Pack RC1 do sieci, a już dzisiaj dowiadujemy się, że firma Microsoft udostępniła publiczną wersję dodatku Service Pack dla systemów Windows 7 i Windows Server 2008.  Service Pack 1 dla Windows 7 nie zawiera żadnych nowych funkcji. Jest to jedynie paczka dotychczas opublikowanych aktualizacji i poprawek do najnowszego systemu giganta z Redmond. Wydanie serwerowe wzbogacono o takie funkcje jak Dynamic Memory i RemoteFX. Do instalacji dodatku SP1 dla wersji serwerowej niezbędny jest system w wersji RTM (Release to Manufacturing). W tym miejscu można pobrać najnowszy Service Pack. Aby to zrobić należy przejść weryfikację legalności systemu operacyjnego. Producent udostępnił obraz ISO (1875.2 MB) oraz instalatory dla systemów 32- i 64-bitowych (514.7 MB i 865.4 MB). [ź] windows7.pl
